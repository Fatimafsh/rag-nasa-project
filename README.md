# NASA RAG Project

This project demonstrates a **Retrieval-Augmented Generation (RAG)** pipeline using Python, Llama Index, and TruLens.

## Features
- Ingest PDF documents (NASA reports)
- Retrieve relevant content using advanced techniques
- Evaluate answers with **RAG Triad metrics**: Context Relevance, Groundedness, Answer Relevance

## How to Run
1. Clone the repository
2. Install dependencies
3. Run `app.py` locally
